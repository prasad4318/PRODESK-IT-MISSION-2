CashFlow – Salary & Expense Tracker
CashFlow is a modern and professional web-based application designed to help users track their salary, manage expenses, and visualize financial data with real-time analytics.
Features
•	Add and manage total salary
•	Track daily expenses with categories
•	Interactive doughnut chart (Chart.js)
•	Real-time balance and expense calculation
•	Transaction history with delete option
•	LocalStorage support (data persists after refresh)
•	Clean and modern UI (Glassmorphism design)
•	Fully responsive design
Technologies Used
•	HTML5 - Structure
•	CSS3 – Styling (Glass UI + Responsive Design)
•	JavaScript  – Functionality
•	Chart.js – Data visualization
•	Lucide Icons – Icons
